HTML Basics

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages.

HTML elements are the building blocks of HTML pages. They are represented by tags.

HTML tags are used to create elements. Tags are enclosed in angle brackets.

For example, the <p> tag is used to create a paragraph.

HTML documents are made up of elements. Each element has a start tag and an end tag.

For example, the <p> tag is a start tag and the </p> tag is an end tag.

HTML elements can also have attributes. Attributes provide additional information about an element.

For example, the <a> tag is used to create a hyperlink. The href attribute specifies the URL of the page the link goes to.

HTML is a markup language, not a programming language. It is used to structure content on the web.
